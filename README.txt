LAB FINAL EXAM PREPARATION FILES
--------------------------------

This folder contains all the labs you requested for your exam preparation.

STRUCTURE:
- Lab01/ : Basic Login + Register (HTML/JS)
- Lab02/ : Validation + Sanitization (HTML/JS)
- Lab03/ : Hashing + RBAC + Cookies (HTML/JS + CryptoJS)
- Lab04/ : Safe Error Handling (HTML/JS)
- Lab05/ : Encryption Admin Only (HTML/JS + CryptoJS)
- Lab06/ : SQL Injection (Node.js + MySQL code)
- Lab07/ : Secure Password Storage with Salt (HTML/JS + CryptoJS)
- Lab08/ : XSS (Node.js + MySQL code)
- Lab09/ : CSRF (Node.js + Express)
- Lab10/ : Secure API + JWT (Node.js + Express)

HOW TO USE:
1. For Labs 01-05 and 07:
   - Open the folder in VS Code.
   - Right-click *.html and select "Open with Live Server".

2. For Labs 06, 08, 09, 10:
   - Open the folder in terminal.
   - Run `npm install` (see individual README.txt in each folder for specific packages).
   - Run `node server.js`.
   - Open browser at http://localhost:3000.
